# Products (frontend)

Páginas y servicios del catálogo de productos/servicios por tenant. Hub principal del pivot producto-first.

## Rutas

- `/products` — listado y acciones rápidas
- `/products/new` — alta de producto
- `/products/:id` — edición

## API

Ver `src/services/products.ts` (`GET/POST/PATCH /products`, `POST /products/bulk`).
