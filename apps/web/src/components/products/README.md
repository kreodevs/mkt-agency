# Product components

Componentes UI reutilizables del eje producto-first.

| Componente | Uso |
|---|---|
| `ProductContextBanner` | Muestra en agentes sobre qué producto trabajan |
| `ProductKeywordSuggestion` | URL del producto + scraping + generación semántica de tags |
| `ProductKeywordTagsInput` | Editor de tags (mín. 3) en wizard de producto |
